# shareframe

Server side of Shareframe: a web platform for sharing photos to networked
e-paper picture frames. Users upload and manage images through the web app;
paired frames receive them over the network and show them as a slideshow.

Related repos:

- [shareframe-board](https://github.com/JOMI195/shareframe-board) — C++ firmware and local web dashboard
- [shareframe-hardware](https://github.com/JOMI195/shareframe-hardware) — Buildroot project that builds the SD card image
